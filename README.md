
#  🍴Sistema de Gestión de Restaurante
## 📌 Descripción
El **Sistema de Gestión de Restaurante** es una aplicación diseñada para facilitar la administración de las operaciones de un restaurante.
Permite gestionar pedidos, menú, inventario y facturación, además de generar reportes para el control del negocio.

El sistema cuenta con dos tipos de usuarios principales: Mesero y Gerente, cada uno con diferentes funcionalidades según su rol dentro del restaurante.

## 🎯 Objetivo del Proyecto
Desarrollar un sistema que optimice la gestión del restaurante permitiendo:

* Registrar pedidos de clientes.
* Administrar el menú del restaurante.
* Controlar el inventario de productos.
* Generar facturas automáticamente.
* Obtener reportes de ventas.
  
# 👥 Roles del Sistema
## 🧑‍🍳 Mesero
El mesero interactúa con el sistema para gestionar los pedidos de los clientes.
**Funciones:**
Iniciar sesión en el sistema.
Ver el menú disponible.
Ordenar platos para los clientes.
Generar facturas de los pedidos.

## 👨‍💼 Gerente
El gerente tiene control administrativo sobre el sistema.
**Funciones:**
Iniciar sesión en el sistema.
Gestionar el inventario del restaurante.
Agregar, modificar o eliminar platos del menú.
Gestionar facturas.

## Generar reportes:
* 📅 Reportes diarios
* 📅 Reportes semanales
* 📅 Reportes mensuales

# ⚙️ Funcionalidades del Sistema

🔐 Autenticación de usuarios (Mesero / Gerente)
📋 Visualización del menú
🍝 Gestión de pedidos
🧾 Generación de facturas
📦 Gestión de inventario
🍽️ Administración del menú
📊 Generación de reportes de ventas

# 🏗️ Estructura del Sistema
El sistema está dividido en varios módulos:
**Autenticación**→ Inicio de sesión de usuarios
**Menú** → Visualización y gestión de platos
**Pedidos** → Registro de órdenes de clientes
**Facturación** → Generación y gestión de facturas
**Inventario** → Control de productos disponibles
**Reportes** → Estadísticas de ventas

# 💻 Tecnologías Utilizadas
Frontend: HTML, CSS, JavaScript
Backend:JDK Spring Boot
Base de datos: MariaDB
Control de versiones: Git y GitHub

 # 📊 Beneficios del Sistema

Mejora la organización del restaurante.
Reduce errores en los pedidos.
Facilita el control de inventario.
Permite analizar las ventas mediante reportes.
